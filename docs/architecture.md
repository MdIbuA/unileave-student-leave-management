# UniLeave Architecture

## Technology Stack

- Oracle APEX 26.1
- Oracle Database
- SQL
- PL/SQL

## Modules

### Home
Landing page with application overview.

### Apply Leave
Students submit leave requests.

### My Leave Requests
Students track submitted requests and status.

### Manage Requests
Faculty and administrators review requests.

### Dashboard
Analytics and reporting.

## Workflow

Student
   ↓
Apply Leave
   ↓
Faculty Review
   ↓
Approve / Reject
   ↓
Status Updated
   ↓
Student Tracking
